# tests/outputs/

This subfolder collects generated diagnostic visuals (15×6 grids, heatmaps, etc.) from simulation pipeline runs.
These are primarily for internal calibration, Rule-13 proxy testing, zone-splatter validation, and reproducibility checks.

- Not canonical manuscript figures.
- Regenerate via `simulate.py` batch mode or notebooks/ scripts as needed.
- Current contents: NK vs Ants, USA vs PRC, Venice probes, etc.

Feel free to add subfolders (e.g., comparatives/, historical/) for better organization later.
