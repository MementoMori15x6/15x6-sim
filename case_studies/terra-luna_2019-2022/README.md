# Terra/Luna Case Study (2019–2022)

This folder contains the full Board scoring and Monte Carlo analysis for Terra/Luna across four stages.

## Key Files
- notebooks/terra-luna_mc_sensitivity.ipynb → Monte Carlo baseline & shock tests
- data/terra_luna_stage3-parabolic-phase_2021.csv → consensus scores
- figures/ → Fracture Meter gauge, timeline, histograms
- results/ → exported tables and PDFs

## How to Reproduce
1. Open terra-luna_mc_sensitivity.ipynb in Colab
2. Run all cells
3. Outputs: collapse probabilities, CIs, and perturbed Fracture Meter distributions
