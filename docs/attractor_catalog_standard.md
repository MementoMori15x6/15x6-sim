# Attractor Catalog Standard – Locked December 2025

The 10 canonical attractors are fixed as follows:

1. Mutualism Ellipsoid  
2. Exploitationism Sinkhole  
3. Competition Toroid  
4. Amensalism Channel  
5. Neutralism Void  
6. Commensalism Plains  
7. Rigid Trap  
8. Boundary Saturation  
9. Chaos Boundary  
10. Stable Corridor (preferred resilience zone)

Deprecated / legacy terms to replace on sight:  
- Parasitism Sinkhole → Exploitationism Sinkhole  
- Hybrid Lattice Band → Stable Corridor  
- Drift Corridor → remove or absorb into Neutralism Void / Commensalism Plains  

Use this list verbatim in all manuscript sections, ledger rows, figures, README, primers, and PRs.
