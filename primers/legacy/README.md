# Primers – Version History

This folder archives the exact prompt versions used across the manuscript and scoring runs.

- **v1.3 Workshop** (`primer_v1.3_workshop.md`): From second LLM scoring workshop rounds. Used for early manuscript scores (Ch 2 Roman cases, Ch 6 crypto, initial Ch 7 baselines). No CRL, no RIT, basic splatter/longevity.  
- **v1.4 Elasticity Patch** (`primer_v1.4_elasticity_patch.md`): Introduced full Rigidity Coefficient, nonlinear penalties, hardened shims. Used for mid-manuscript scores (late Ch 2 drifts, Ch 7 shocks). Pre-RIT.  
- **v1.5 Elasticity + Invariance Patch** (`primer_v1.5_elasticity_invariance_patch.md`): Added Replicator Invariance Test (RIT) and Anchor-First scoring (variance stabilization). Used for refined mid-to-late scores (Ch 7 shocks, Ch 10 what-ifs, early Ch 12 rows). Pre-Exploitationism.  
- **v1.6 Clean** (`primer_v1.6_clean.md`): Tightened layout, model identifier, Exploitationism terminology swap.  
- **v1.7 Baseline** (`primer_v1.7_baseline.md`): Final locked version (Dec 2025) with CRL anchors enforced and proxy tuned.

**Manuscript scoring note**  
Most scores prior to Chapter 9 CRL lock (v1.2–v1.5) were generated with unconstrained primers (hinge metrics subject to model drift). The Late VOC consensus row (Ch 12) and future ledger entries use v1.7 enforced anchors. Legacy scores are illustrative; re-run with v1.7 for direct comparison.
