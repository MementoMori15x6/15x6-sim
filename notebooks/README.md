# /notebooks/ – Scoring & Visualization Templates

This folder contains Jupyter notebooks (.ipynb) for interactive use of the compass protocol and figure generation.

## Current Status (March 2026)
- Empty / under construction
- Planned starter notebooks (when added):
  - `score_your_replicator.ipynb` — Step-by-step 35-metric scoring template + run simulate.py
  - `ensemble_run_example.ipynb` — Stochastic drift & longevity fit demo (n=100 runs)
  - `triangulation_plot.ipynb` — Phase-space scatter of the four anchors (USA_1789, USSR_1937, VOC golden/late)
  - `figure_generation_templates.ipynb` — Helpers for Ch 9/11/12 diagrams (heatmaps, drift curves, hazard rates)

## How to Use (once notebooks exist)
1. Open in Colab or local Jupyter:
   - Click "Open in Colab" badge (if added) or upload .ipynb
   - Or local: `jupyter notebook notebooks/score_your_replicator.ipynb`

2. Fill scores → run cells → see X/Y, Rule-13 proxy, state, longevity, and plots

3. Contribute:
   - Create a new notebook for a specific replicator or figure
   - Add Markdown explanations + code cells
   - PR with clear title (e.g. "Add scoring notebook for British East India Company")

These notebooks make the compass more accessible — no git or CLI required for basic scoring or visualization.

Questions? Open an issue or PR in the root repo.
