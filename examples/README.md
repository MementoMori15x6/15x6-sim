## Canonical Examples & Baselines

The `/examples/` folder contains only the five locked canonical input CSVs used for pipeline tuning and manuscript reference:

- `eusocial_ant_colony.csv` — High-trust biological foil
- `influenza_molecular.csv` — Molecular-scale replicator
- `modern_north_korea.csv` — High-parasitism authoritarian proxy (1980–2025)
- `ussr_1917-1991.csv` — Mid-range suppression case
- `united_states_west_1971-2025.csv` — Stressed mutualism / libertarian tilt

All other provisional, historical probes (e.g., Venice), or early drafts are archived in `/tests/inputs/` or `/data/`. Do not add new files here without promoting to canonical status.
